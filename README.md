# Ubuntu Setup Script
Using **deb**, **apt** and **snap** packages to install softwares like:

- SDKMAN (Java 8 and Maven)
- Docker
- NodeJS 12
- Postman
- Dbeaver
- and others...

## Running

```sh
git clone https://github.com/julioomoura/ubuntu-init-script.git
cd ubuntu-init-script
chmod +x script.sh
./script.sh
```
